# Terraform Deployment for Foundry

This is a Terraform infrastructure-as-code repository for Azure that uses the Azure naming module to provision resources like Azure AI Services and Cognitive Deployments with CAF-compliant naming conventions. It's designed as a foundational template for consistent Azure resource deployment.
